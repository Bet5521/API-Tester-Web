# 通用接口调试工具V3.2 - README

## 系统简介

通用接口调试工具是一个功能强大的HTTP接口测试工具，支持多种请求方法，提供直观的用户界面，方便开发人员和测试人员快速测试和调试API接口。

## 主要功能

### 1. 接口测试
- 支持GET、POST、PUT、DELETE等多种HTTP请求方法
- 可添加请求参数、请求体参数和请求头
- 支持JSON格式的请求体编辑
- 实时显示响应状态、响应时间和响应结果
- 请求体参数与JSON区域双向联动

### 2. 接口管理
- 添加、编辑、删除接口
- 导出接口配置为JSON格式
- 导入接口配置
- 三列布局的接口列表，支持多选操作

### 3. 历史记录
- 自动保存请求历史
- 查看历史请求详情
- 清空历史记录

### 4. 系统设置
- 设置协议（HTTP/HTTPS）
- 设置基础URL
- 设置请求超时时间
- 设置最大历史记录数

### 5. 其他特性
- 固定的页头和页脚，提供更好的用户体验
- 响应式设计，适配不同屏幕尺寸
- 请求头参数X-lr-request-id自动使用当前时间
- JSON类型参数支持子列表编辑
- 参数组管理功能，支持数组类型JSON的编辑

## 技术栈

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 4

## 系统要求

- 现代Web浏览器（Chrome、Firefox、Edge等）
- 互联网连接（用于加载Bootstrap等依赖）

## 使用说明

1. 打开系统页面 `index.html`
2. 在"接口测试"页面选择或输入接口信息
3. 添加必要的参数和请求头
4. 点击"发送请求"按钮测试接口
5. 在"接口管理"页面管理您的接口配置
6. 在"历史记录"页面查看过往请求
7. 在"设置"页面配置系统参数
8. 在"操作手册"页面查看详细使用指南
